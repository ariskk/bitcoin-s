  24.  case object Broadcasted extends DLCState with Product with Serializable

  25.  case object Claimed extends DLCState with Product with Serializable

  26.  case object Confirmed extends DLCState with Product with Serializable

    The state where the DLC funding transaction has been confirmed on-chain and no execution paths have yet been initiated.

  27.  case object Offered extends DLCState with Product with Serializable

  28.  case object Refunded extends DLCState with Product with Serializable

  29.  case object RemoteClaimed extends DLCState with Product with Serializable

  30.  case object Signed extends DLCState with Product with Serializable
