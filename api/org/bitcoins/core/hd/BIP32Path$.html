  17.  def fromString(string: String): BIP32Path

    Parses a string representation of a BIP32 path.

    Parses a string representation of a BIP32 path. This is on the form of

    m/level/hardenedLevel'/...

    Where level is an integer index and hardenedLevel is an integer index followed by a '. Different notation is used in BIP32, but this is the most common way of writing down BIP32 paths.

    See also

    BIP43 and BIP44 for examples of this notation.
